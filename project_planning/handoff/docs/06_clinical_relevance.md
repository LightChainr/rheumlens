# 临床意义与转化路径

## 1. 当前临床意义

RheumLens 当前不是临床诊断工具开发项目，而是临床转化前的方法学基础研究。直接价值包括：

1. 判断模型识别的是疾病，还是 donor 泄漏、批次、细胞数量和样本处理；
2. 避免将高队列内 AUC 过早解释为临床泛化；
3. 将单细胞信息压缩为可解释的患者级免疫表型；
4. 建立 foundation model 进入患者分层研究前的评估标准。

## 2. 患者级表型的潜在价值

新方法试图区分：

- 全体细胞轻度 IFN 升高；
- 少数细胞极端 IFN 激活；
- B-cell/plasmablast 丰度异常；
- myeloid inflammatory state；
- T-cell activation 和异质性；
- cell composition 与 within-cell-type transcriptional shift。

这些表型未来可用于：

- 患者亚型划分；
- 临床试验富集；
- 治疗机制研究；
- 纵向疾病监测；
- 治疗反应和复发风险预测。

## 3. 转化所需后续证据

在提出实际临床应用前，需要：

- 多中心、跨平台外部验证；
- SLE 与其他自身免疫病、感染及真实鉴别诊断比较；
- 与补体、抗 dsDNA、SLEDAI 和常规临床指标比较；
- 纵向和治疗前后数据；
- calibration、decision-curve 和临床净获益分析；
- 固定样本处理和可重复检测流程；
- 前瞻性验证。

因此当前最稳妥的表述是：

> RheumLens 建立从海量单细胞数据中提取可靠、可解释、可迁移的患者级免疫表型的方法基础，为未来的患者分层、疗效预测和疾病监测提供候选技术框架，而非证明即时临床诊断可用性。
